# KYNTRA402 - Rewards Platform

Платформа для получения вознаграждений за продвижение токена $KYN402 в X (Twitter).

## Технологии

- React 18
- TypeScript
- Vite
- CSS3 с анимациями

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Собрать для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```

- Вторичный текст: `#9CA3AF` (холодный серый)

## Интеграция с Phantom Wallet

Сайт поддерживает подключение Phantom Wallet:

1. Установите расширение [Phantom Wallet](https://phantom.app/)
2. Нажмите кнопку "Connect Wallet" в header
3. Подтвердите подключение в окне Phantom
4. Адрес кошелька автоматически заполнится в форме отправки промо

При подключении кошелька:
- Кнопка меняет цвет на зелёный
- Отображается сокращённый адрес (первые 4 и последние 4 символа)
- При наведении на подключенную кнопку она становится красной (для отключения)
- Адрес кошелька автоматически заполняется в форме
